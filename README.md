# 산속농원 사과 판매 사이트

제천 산속농원의 고객용 사과 판매 페이지와 판매 관리 페이지입니다.

## 페이지

- `index.html`: 고객용 판매 사이트
- `admin.html`: 상품·주문·판매 정보 관리

## 실행

정적 웹사이트이므로 `index.html`을 열거나 정적 웹 서버에서 바로 실행할 수 있습니다.

## 데이터 저장 방식

상품, 주문, 판매자 정보와 상세 상품 사진은 Firebase Firestore에 저장되며, 브라우저 저장공간은 임시 캐시로 사용합니다.

판매 관리 페이지에는 임시 암호 화면이 적용되어 있습니다. 실제 운영 단계에서는 Firebase Authentication 기반 관리자 인증으로 교체해야 합니다.
